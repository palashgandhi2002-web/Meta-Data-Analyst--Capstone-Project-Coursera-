# Statistical Analysis of Ad Campaign Performance: Facebook vs. AdWords

## Project Overview

This project investigates the effectiveness of Facebook Ads versus Google AdWords campaigns using statistical analysis techniques. By leveraging data on ad clicks and conversions, the analysis determines which platform offers superior performance and provides actionable insights to optimize future digital marketing strategies.

## Key Features

- Comparative analysis of Facebook and AdWords ad campaigns using real data
- Calculation of descriptive statistics (mean, median, mode, standard deviation)
- Data visualization through histograms, scatter plots, and contingency tables
- Hypothesis testing (T-test) to evaluate statistical significance
- Regression modeling to predict conversions based on ad clicks
- Clear, actionable recommendations for marketing budget allocation

## Tech Stack and Tools

- PowerPoint (presentation and data visualization)
- Google Sheets/Excel (data handling and calculations)
- Statistical methods (T-test, regression analysis)
- Data analytics lifecycle principles (OSEMN framework)

## Project Structure

- `Statistics-Foundations-Capstone-Deck-TEMPLATE.pptx`: The main presentation deck documenting the analysis, visualizations, results, and recommendations.

## Insights and Outcomes

- Facebook Ads generated significantly more conversions per click compared to AdWords within the evaluated dataset.
- Hypothesis testing confirmed that the difference was statistically significant.
- Regression analysis showed a strong correlation between Facebook ad clicks and conversions, suggesting high predictability and effectiveness.
- Recommendations include increasing investment in Facebook Ads and further segmenting data by creative types or audience demographics for deeper insights.
